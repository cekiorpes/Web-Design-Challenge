Summary: Latitude vs X

This project examined how particular weather parameters changed for a city based on the proximity or distance to the equator. The cities were randomly chosen and weather data, including country, latitude, longitude, maximum temperature, humidity, cloudiness, and wind speed were obtained from OpenWeatherMap API for the day of query (January 25th, 2017).

These data were then plotted using Matplotlib to graphically evaluate the results and determine whether or not a potential correlation exists. These images display the results.
